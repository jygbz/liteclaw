透明可控的 AI Agent 基础框架

LiteClaw 是一个面向企业场景的 AI Agent 执行框架。它通过 LangGraph 状态机、预检式工具调用、分层记忆与纵深防御沙盒，让 AI 的每一次操作都可观测、可审计、可回滚。

核心特性

🛡️

预检式工具调用

工具执行前 LLM 先阅读完整说明书进行风险评估，确认安全后再执行。避免名称迷惑、参数误用导致的安全事故。

🧠

分层记忆架构

长期画像（Markdown）与短期摘要（State）分离，40 轮自动触发按回合裁剪，支持跨会话复用与长对话稳定性。

🔒

纵深防御沙盒

路径规范化、5 条正则熔断、60 秒超时兜底，三层防御拦截路径穿越、危险命令与长时间挂起。

📋

异步审计日志

单例 + 内存队列 + 守护线程异步写入，5 类事件全记录，对主流程零阻塞，满足企业可观测性要求。

⚡

懒加载技能引擎

启动仅解析前 50 行元数据，首次调用才加载完整内容，LRU 缓存 + mtime 热更新，万级技能也能秒启。

🔌

多模型 Provider 工厂

统一封装 OpenAI / Anthropic / 本地模型等接口，通过配置切换，降低多模型接入成本。

系统架构

LiteClaw 采用四层架构：入口层负责交互，引擎层承载 Agent 核心循环，工具层暴露安全执行环境，运行时层提供持久化、审计与调度。

核心执行流程：用户输入 → 审计记录 → 上下文裁剪与记忆注入 → LLM 决策 → 若需工具则路由到 Tools 节点 → 沙箱执行 → 结果回写 → 循环直到输出最终答案。整个状态由 AsyncSqliteSaver 持久化，支持断点续传。
